# Welcome to Looopy AI

Looopy AI is a reactive, extensible AI agent framework built with TypeScript and RxJS. It provides a powerful, stream-based architecture for creating advanced AI agents that can handle complex, multi-turn conversations and execute tasks in real-time.

## Key Features

- **Reactive Architecture**: Built on RxJS, Looopy AI uses observables to manage asynchronous operations and data streams, making it easy to create complex, event-driven agents.
- **Extensible**: The framework is designed to be highly extensible, with a modular architecture that allows you to easily add new providers, stores, and tools.
- **Observable**: Looopy AI includes built-in support for OpenTelemetry, making it easy to trace and monitor the execution of your agents.
- **Scalable**: The framework is designed to be scalable, with support for distributed systems and a variety of persistence layers.
